# 🚀 FireRed React App

A simple responsive React application built using **React + Vite + React Router**.  
This project includes reusable components, multiple pages, and a responsive layout.

---

## 📁 Project Structure

src/
components/
Navbar.jsx
Hero.jsx
Card.jsx
Footer.jsx

pages/
Home.jsx
About.jsx
Dashboard.jsx
Login.jsx

App.jsx
main.jsx
styles.css

---

## ✨ Features

- ⚛️ Built with React (Vite)
- 🔀 React Router for navigation
- 🧩 Reusable components (Navbar, Hero, Card, Footer)
- 📱 Fully responsive design (mobile-friendly)
- 🏠 Home page with card grid layout (minimum 2+ cards)
- 📄 3 additional pages (About, Dashboard, Login)
- 🎯 Clean and simple UI

---

## 🖥️ Pages

- `/` → Home Page
- `/about` → About Page
- `/dashboard` → Dashboard Page
- `/login` → Login Page
